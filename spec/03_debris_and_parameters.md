# ゴミとパラメータ一覧

本作に登場する各種ゴミのビジュアル定義および、ゲーム内で用いられる各種数値パラメーターの一覧です。

## 1. 数値パラメータ定義

| パラメータ名 | 設定値 | 説明 |
| :--- | :--- | :--- |
| **制限時間（1日あたり）** | 1分 (60秒) | 1ステージ（1日）のプレイ可能時間。 |
| **ゴミの蓄積上限数** | 30個 | 画面内に溜まることができるゴミの最大数。30個に達すると強制リザルト（クリア失敗）となる。 |
| **初期ゴミ落下間隔** | 例：約4.0秒 | ステージ開始時のゴミが降ってくる基本間隔。 |
| **落下間隔の最小値** | 例：約2.5秒 | 時間経過によって徐々に短縮される落下間隔の限界値（少しだけのスピードアップ）。 |
| **基本給料（分別1個正解）** | 90 | ゴミを正しく分別するごとに獲得できる金額。 |
| **ミスペナルティライン** | 3回 | 1ステージ内で合計3回誤った分別を行うと、その日の給料が0になる。 |
| **生活費（1日目終了時）** | 0 | 1日目終了時は生活費は発生しません。 |
| **生活費（2日目終了時）** | 50 | 2日目の終了時に支払う必要がある金額。支払えない場合はゲームオーバー。 |

## 2. ゴミのビジュアルバリエーション

落下してくるゴミには、以下のビジュアルバリエーションが存在し、毎回ランダムに生成・決定されます。

### 一般のゴミ（1日目から登場）
プレイヤーはこれらのゴミを「観察」し、表面または裏面にあるマークを確認して分別します。

| ゴミの名称 | 想定されるビジュアル特徴 | 燃える/燃えない |
| :--- | :--- | :--- |
| **紙くず** | クシャクシャに丸められた白い紙。 | マークに従う |
| **ペットボトル** | 透明で少し潰れたプラスチックボトル。 | マークに従う |
| **りんごの芯** | 食べかけで少し茶色く変色したりんごの芯。 | マークに従う |
| **お菓子の袋** | カラフルでシワのあるアルミ加工された袋。 | マークに従う |
| **新聞紙** | 折りたたまれて少し破れた白黒の新聞。 | マークに従う |
| **謎の四角いブロック**| 人工物か自然物か判別できない、無機質な立方体。 | マークに従う |
| **謎の球体** | 表面に細かい凹凸がある、奇妙な質感の丸い物体。| マークに従う |
| **魚の頭** | 骨が見えかけている魚の頭部の生ゴミ。 | マークに従う |

*※一般のゴミは、出現時に「燃えるゴミのマーク」または「燃えないゴミのマーク」のいずれか一方が、表側か裏側のランダムな位置に1つ配置されます。*

---

### 特殊なゴミ「骨」（2日目から登場）
2日目から「骨」を模したゴミが新しく出現します。

| ゴミの名称 | 想定されるビジュアル特徴 | 判別基準・マークの有無 | 正解の分別ボックス |
| :--- | :--- | :--- | :--- |
| **本物の骨** | 白く乾燥した、典型的なT字型または棒状の動物の骨。 | **マークなし** (表裏のどちらにもマークは存在しない) | **骨ボックス** |
| **プラスチック製の骨** | 本物の骨と似ているが、やや人工的な光沢や継ぎ目がある骨。 | **燃えないゴミのマークあり** (表裏のどちらか一箇所に付いている) | **燃えないゴミボックス** |
