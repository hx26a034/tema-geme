# ゲーム概要

本作は、落ちてくるゴミを正しく分別して給料を稼ぎ、日々の生活費を補っていく2Dパズルシミュレーションゲームです。プレイヤーは日にちを重ねるごとに増えていく複雑な分別のルールに対応しながら、制限時間内に多くのゴミを正しく仕分けることを目指します。

## 基本仕様
- **ジャンル**: 2Dパズルシミュレーション
- **画面構成**: 1画面に収まるサイズ（シングル画面構成）
- **進行方式**: 日にち制（デイリーサイクル）
- **プレイ制限時間**: 1日あたり1分（60秒）
- **ゲームの終了条件**:
  - **ゴミの許容量オーバー**: 画面内のゴミが30個に達した場合（その日はクリア失敗となり、給料は支給されません）。
  - **生活費の滞納（ゲームオーバー）**: 2日目の終了時点で必要な生活費（50）を支払えない場合（1日目は猶予されますが、2日目終了時点で支払えなければゲームオーバーとなります）。
